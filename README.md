# CineNova - Кинотеатр нового поколения

Инновационная веб-платформа для бронирования билетов в кинотеатрах с интеграцией элементов ИИ, модульной системой раздельной оплаты (Split-Pay) и интерактивными профилями пользователей.

## Структура проекта
- `frontend/` — Клиентская часть (HTML, CSS, клиентские скрипты UI и API).
- `backend/` — Серверное приложение на Node.js и Express.
- `backend/database/` — Инициализация и подключение локальной базы данных SQLite.

## Системные требования
- Node.js версии 18.0.0 или выше.
- Пакетный менеджер npm.

## Локальный запуск приложения
1. Установите зависимости проекта, выполнив в терминале корневой директории:
   npm install
2. Выполните первичную инициализацию таблиц базы данных и тестовых данных:
   npm run seed
3. (Опционально) Выполните синхронизацию каталога фильмов:
   npm run sync
4. Запустите локальный сервер:
   npm start
5. Откройте в браузере страницу: `http://localhost:3000`

## Развёртывание на платформу Render
Приложение полностью подготовлено к публикации на хостинге Render. Чтобы база данных SQLite не сбрасывалась при перезапусках контейнера, настройте постоянный диск (Persistent Disk):

1. Создайте новый **Web Service** на Render и подключите свой Git-репозиторий.
2. Установите следующие параметры конфигурации:
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `npm run seed && npm start`
3. Перейдите в раздел **Advanced**, нажмите **Add Environment Variable** и добавьте:
   - `RENDER` = `true`
4. Спуститесь к блоку **Disks** и нажмите **Add Disk**:
   - **Name**: `cinenova-data`
   - **Mount Path**: `/data`
   - **Size**: `1 GiB`
5. Сохраните изменения и запустите процесс развёртывания (Deploy).
